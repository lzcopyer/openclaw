# Flowchart Patterns

## Standard Flowchart Symbols

| Symbol             | Shape                | Meaning                         |
| ------------------ | -------------------- | ------------------------------- |
| Start/End          | Oval                 | Beginning or end of process     |
| Process            | Rectangle            | Action or operation             |
| Decision           | Diamond              | Yes/No or True/False branch     |
| Input/Output       | Parallelogram        | Data input or output            |
| Connector          | Circle               | Connect to another part of flow |
| Document           | Rectangle with wave  | Document or report              |
| Predefined Process | Rectangle with lines | Subroutine or function          |

## Flowchart Best Practices

1. **Direction:** Flow from top to bottom or left to right
2. **Labels:** Label all decision branches (Yes/No, True/False)
3. **Consistency:** Use standard symbols consistently
4. **Simplicity:** Break complex flows into sub-processes
5. **Entry/Exit:** Every flowchart should have one START and one END
